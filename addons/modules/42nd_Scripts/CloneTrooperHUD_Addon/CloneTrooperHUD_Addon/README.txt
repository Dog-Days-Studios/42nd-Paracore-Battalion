Clone Trooper HUD Addon
=======================

What it does:
- Enemy diamond markers
- Current weapon display
- Magazine / reserve round display
- Lower-left minimap
- User-adjustable colors
- Colorblind presets
- Optional helmet-restricted display
- CBA settings support

How to pack it:
1. Put the CloneTrooperHUD_Addon folder in your Arma 3 work drive.
2. Pack it with Addon Builder or your preferred PBO tool.
3. Load it with CBA_A3.

Required mod:
- CBA_A3

Recommended next steps:
- Replace the default enemy icon with a custom holographic diamond texture.
- Add a helmet overlay picture control for a full clone visor look.
- Restrict enemy markers to detected / known targets only if you want less arcade behavior.
- Add custom sound / warning states for low ammo or critical damage.

CBA settings notes:
- Enable Clone HUD: master toggle
- Only With Clone Helmet: restricts HUD to listed helmets
- Clone Helmet Classnames: comma-separated class list
- Enemy Marker Distance: max marker range
- Show Mini Map: toggles minimap
- HUD Color Preset: includes colorblind presets
- Custom HUD Color: used when preset is Custom
